Welcome to your mendelson AS2 web interface

Congratulations - your mendelson AS2 web interface is running fine.

It is possible to customize the welcome screen as seen here.



How could you customize this text?

Please navigate to your mendelson AS2 installation directory and open the file "webas2_welcome.html" in a normal text editor or a HTML editor to customize this text.
The standard HTML tags are allowed there: Links, Text attributes (e.g. ABC) and much more. To make the changes valid please restart the mendelson AS2.

How do you disable the webclient?

Please move away the file <mendelson AS2 installation>/jetty9/webapps/webas2.war, then restart the mendelson AS2 and you are done.
