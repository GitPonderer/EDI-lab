Proxy

Der mendelson opensource AS2 Server benötigt eine direkte Verbindung zum Internet. Wenn Sie keine direkte Verbindung zur Verfügung stellen können, sondern nur über einen Proxy nach aussen komminizieren können, müssen Sie jenen hier eintragen.

MDN

Stellen Sie hier ein, wie lange das System auf eine asynchrone MDN warten soll, bevor die Nachricht als fehlerhaft markiert wird. Das sind in der Regel 30 Minuten.

Sicherheit

Die Zertifikate/Schlüssel zur Sicherung der Verbindung (ssl) und die Zertifikate zum Signieren und Verschlüsseln der Nachrichten sind voneinander getrennt. Bitte geben Sie zum Senden von Nachrichten über Https hier den Pfad zum Keystore mit den Zertifikaten für die ssl Verbindung an.

Verzeichnisse

Hier können Sie das Basisverzeichnis für die ein- und ausgehenden Nachrichten einstellen. Dies steht standardmässig auf dem Unterverzeichnis "messages" im as2 Verzeichnis.

Systempflege

Sie können hier einstellen, wie lange Nachrichten (ohne Beachtung ihres Status) auf dem System gehalten werden sollen. Ist diese Option eingeschaltet, werden Nachrichten, die älter als die einstellte Anzahl von Tagen sind, gelöscht.

Sprache

Wählen Sie die Sprache der Ausgaben und der GUI. Diese Einstellung wird erst nach dem nächsten Starten des Servers gültig. Es werden im Moment die Sprachen Deutsch und Englisch unterstützt.
