Anbindung

Der as2 Server kann einfach an weiterverarbeitende System angebunden werden. Wir empfehlen dazu die mendelson business integration. Sämtliche eingehende und ausgehende Nachrichten werden in einer Verzeichnisstruktur gehalten. Alle im folgenden genannten Sende- und Empfangsverzeichnisse beziehen sich auf das konfigurierbare Unterverzeichnis zum Verarbeiten von Nachrichte, das standardmaessig auf "messages" steht.
Zum Versenden von Daten müssen diese nur in das entsprechende Partnerverzeichnis gelegt werden. Die Partnerdaten für jedes Verzeichnis sind hinterlegbar, sodass automatisch verschlüsselt, signiert und versandt wird.
Der Empfang von Daten geschieht automatisch.

Standardmässig sind folgende URLs zum Empfang definiert:
http://<host>:8080/as2/HttpReceiver
https://<host>:8443/as2/HttpReceiver

Partnerverzeichnisse

Unterhalb des Nachrichtenverzeichnisses wird für jeden Partner folgende Verzeichnisstrukt angelegt:

<Partnername>

<Partnername>/inbox
Wenn Nachrichten eingehen, werden die Payloads hier gespeichert.

Dieses Verzeichnis wird nur für lokale Stationen angelegt.

<Partnername>/outbox/<Lokale station>
Sollen Nachrichten an eine entsprechenden Partner versandt werden, müssen sie hier hineingelegt werden.
Dieses Verzeichnis wird nicht für lokale Stationen angelegt.

<Partnername>/error/<Lokale station>/<Datum>
Kommt es zu einem Fehler beim Verpacken der Nachrichten, wird die Nachricht hierher verschoben.
Diese Verzeichnisse werden nicht für lokale Stationen angelegt.

<Partnername>/sent/<Lokale station>/<Datum>
Wurde eine Nachricht erfolgreich versandt, werden die entsprechenden Daten verpackt im Rohformat hier abgelegt.
Dieses Verzeichnis wird nicht für lokale Stationen angelegt.
