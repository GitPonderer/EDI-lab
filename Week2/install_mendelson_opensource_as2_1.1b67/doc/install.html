mendelson opensource AS2 server Installation Instructions

Software requirements

Please remember that all needed software needed to run the AS2 server is shipped with the mendelson opensource AS2 (full installer only). It is not necessary to download any additional software from other sites.
Anyway mendelson opensource AS2 includes other third party software. Please refer to the license directory to have a look at the license files of included third party software.

The mendelson opensource AS2 server will not run without JVM 1.7 or higher. The installation setup of the mendelson software includes the JVM. If the installer asks you which VM to install it is recommended to install the bundled VM.
The mendelson opensource AS2 server will run with both 32 bit and 64 bit VMs.

Installation

For the installation just start the shipped installer. If the installer doesn't work for you please contact the mendelson support at support@mendelson.de.

Starting mendelson opensource AS2

The mendelson opensource AS2 server comes with an embedded HTTP server that is automatically started on program start.
The URL the as2 server will listen to by default is http://localhost:8080/as2/HttpReceiver.
You could point your browser to that URL after starting up the mendelson opensource AS2 server to check if everything is ok.
The port for ssl connections is by default 8443. SSL is not enabled by default, you have to modify the file jetty/etc/jetty.xml if you want to use it.

Non-Windows OS (linux, solaris, mac OS)

The installer package for windows includes everything that is needed if you want to start the server. It is recommended to use the .exe installer for windows os.
If you used the zip (all platform) installer, you have to install some platform specific software:

  1. First you have to download and install a JRE. You could have a look at http://www.oracle.com/technetwork/java/javase/downloads/index.html for that (Java SE Downloads).
  2. If you live outside the US you have to install the full crypto strength package to the VM. You could get it from sun/oracle, too. (Java Cryptography Extension (JCE) Unlimited Strength Jurisdiction Policy Files 7). Without installing this package (just copy the included files into jre/lib/security of the VM) the mendelson opensource AS2 server will not (!) start.
  3. There are some start scripts for several OS included. You could have a look on them if one of them matches your OS. Mainly the .sh script should work on all unix systems and the .bat script should work on all windows systems.
